cik
===
**Des**:  
*cik* is a command-line bitcoin wallet.


**Requirments:**  
requires docopt:  
	```pip install docopt```


**USAGE:**  
	cik.py reset  
	cik.py init ([-]| --addr=<wallet_address>)  
	cik.py status  
	cik.py (-h | --help)  
	cik.py --version  
**Options:**  
  -h --help     Show this screen.  
  --version     Show version.
